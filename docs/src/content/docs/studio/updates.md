---
title: Updating Studio
description: How to check for and install a new version of Doodle Studio.
---

Doodle Studio lets you know when a new version is available. It never downloads
or installs updates on its own.

## When an update is available

Studio checks for new versions when it starts. If you already have the latest
version, nothing appears.

When a newer version is available, Studio shows its version alongside the one
you have:

> Version 0.3.0 is available. You have 0.2.0.

Select **Download** to open the download in your browser. Studio stays open, so
you can install the update whenever you are ready.

## Check for updates yourself

You can check at any time:

- Open **Help → Check for Updates…**
- Or open the command palette with Cmd/Ctrl+K and choose **Check for updates**

Studio will tell you whether it is up to date. If it cannot check, you can
select **Try Again**.

## Install the update

After the download finishes:

- **Windows:** run the downloaded installer. It replaces your current version
  of Studio.
- **macOS:** open the downloaded disk image and drag Doodle Studio into your
  Applications folder. When asked, choose to replace the existing app.

Open Studio again afterward. You can find the installed version under
**Help → About Doodle Studio**.

## Security prompts

Because Studio is distributed without a paid signing certificate, your
operating system may ask you to confirm that you want to open it.

- **Windows:** if you see **Windows protected your PC**, select **More info**,
  then **Run anyway**.
- **macOS:** if macOS says it could not verify the app, open **System Settings
  → Privacy & Security** and select **Open Anyway**.

You may need to confirm this again after installing a new version.

## Your projects are safe

Updating Studio does not change your game files or the Doodle Engine version
used by your project. Your projects remain exactly where they are.
