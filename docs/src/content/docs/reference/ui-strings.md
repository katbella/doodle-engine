---
title: UI Strings
description: Localization keys and English defaults used by the built-in renderer.
---

The built-in renderer uses these `ui.*` keys for buttons, menus, headings, status messages, credits, and time labels. Add a key to a locale file when you want to translate or replace its English default.

Placeholders in curly braces are replaced while the game runs. Keep the same placeholder in a translation.
